更改Mac OS X登陆密码

重设Mac OS X开机密码

  1. 关闭 Mac。如有需要，请按住电源按钮直到 Mac 强制关机。
  2. 按（但不用按住）电源按钮启动 Mac，然后立即按住键盘上的 Command-R。
  3. 看到 Apple 标志出现后，您可以松开在上一步中按住的按键。
  4. 恢复系统屏幕出现后，请选取“实用工具”>“终端”。
  5. 在“终端”中，输入此命令并按下 Return 键以打开“重设密码”实用工具：
    resetpassword
  6. “重设密码”实用工具窗口应会出现。
